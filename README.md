# Manim Video Creator Plugin Marketplace

Claude Code用のManimアニメーション動画作成プラグイン。

## インストール

```bash
claude /plugin https://github.com/ymmt/cc-manim-video-creator-plugin
```

## 含まれるプラグイン

- **manim-video-creator**: Manim（Mathematical Animation Engine）を使用してTTSナレーション・BGM付きのアニメーション動画を作成

詳細は [plugins/manim-video-creator/README.md](plugins/manim-video-creator/README.md) を参照してください。
